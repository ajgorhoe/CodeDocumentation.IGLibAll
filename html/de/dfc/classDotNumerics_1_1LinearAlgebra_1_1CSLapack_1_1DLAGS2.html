Detailed Description

– LAPACK auxiliary routine (version 3.1) – Univ. of Tennessee, Univ. of California Berkeley and NAG Ltd.. November 2006

Purpose

DLAGS2 computes 2-by-2 orthogonal matrices U, V and Q, such that if ( UPPER ) then

U'A*Q = U'( A1 A2 )*Q = ( x 0 ) ( 0 A3 ) ( x x ) and V'B*Q = V'( B1 B2 )*Q = ( x 0 ) ( 0 B3 ) ( x x )

or if ( .NOT.UPPER ) then

U'A*Q = U'( A1 0 )*Q = ( x x ) ( A2 A3 ) ( 0 x ) and V'B*Q = V'( B1 0 )*Q = ( x x ) ( B2 B3 ) ( 0 x )

The rows of the transformed A and B are parallel, where

U = ( CSU SNU ), V = ( CSV SNV ), Q = ( CSQ SNQ ) ( -SNU CSU ) ( -SNV CSV ) ( -SNQ CSQ )

Z' denotes the transpose of Z. /summary>
