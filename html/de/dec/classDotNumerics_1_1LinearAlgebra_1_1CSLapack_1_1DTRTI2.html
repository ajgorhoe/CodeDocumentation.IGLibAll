Purpose

DTRTI2 computes the inverse of a real upper or lower triangular matrix.

This is the Level 2 BLAS version of the algorithm.

/summary>

Parameters
UPLO(input) CHARACTER*1 Specifies whether the matrix A is upper or lower triangular. = 'U': Upper triangular = 'L': Lower triangular /param>
Parameters
DIAG(input) CHARACTER*1 Specifies whether or not the matrix A is unit triangular. = 'N': Non-unit triangular = 'U': Unit triangular /param>
Parameters
N(input) INTEGER The order of the matrix A. N .GE. 0. /param>
Parameters
A(input/output) DOUBLE PRECISION array, dimension (LDA,N) On entry, the triangular matrix A. If UPLO = 'U', the leading n by n upper triangular part of the array A contains the upper triangular matrix, and the strictly lower triangular part of A is not referenced. If UPLO = 'L', the leading n by n lower triangular part of the array A contains the lower triangular matrix, and the strictly upper triangular part of A is not referenced. If DIAG = 'U', the diagonal elements of A are also not referenced and are assumed to be 1.
On exit, the (triangular) inverse of the original matrix, in the same storage format. /param>
Parameters
LDA(input) INTEGER The leading dimension of the array A. LDA .GE. max(1,N). /param>
Parameters
INFO(output) INTEGER = 0: successful exit .LT. 0: if INFO = -k, the k-th argument had an illegal value /param>
