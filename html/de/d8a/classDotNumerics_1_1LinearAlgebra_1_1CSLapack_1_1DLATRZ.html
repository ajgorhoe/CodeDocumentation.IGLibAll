Detailed Description

– LAPACK routine (version 3.1) – Univ. of Tennessee, Univ. of California Berkeley and NAG Ltd.. November 2006

Purpose

DLATRZ factors the M-by-(M+L) real upper trapezoidal matrix [ A1 A2 ] = [ A(1:M,1:M) A(1:M,N-L+1:N) ] as ( R 0 ) * Z, by means of orthogonal transformations. Z is an (M+L)-by-(M+L) orthogonal matrix and, R and A1 are M-by-M upper triangular matrices.

/summary>

Purpose

DLATRZ factors the M-by-(M+L) real upper trapezoidal matrix [ A1 A2 ] = [ A(1:M,1:M) A(1:M,N-L+1:N) ] as ( R 0 ) * Z, by means of orthogonal transformations. Z is an (M+L)-by-(M+L) orthogonal matrix and, R and A1 are M-by-M upper triangular matrices.

/summary>

Parameters
M(input) INTEGER The number of rows of the matrix A. M .GE. 0. /param>
Parameters
N(input) INTEGER The number of columns of the matrix A. N .GE. 0. /param>
Parameters
L(input) INTEGER The number of columns of the matrix A containing the meaningful part of the Householder vectors. N-M .GE. L .GE. 0. /param>
Parameters
A(input/output) DOUBLE PRECISION array, dimension (LDA,N) On entry, the leading M-by-N upper trapezoidal part of the array A must contain the matrix to be factorized. On exit, the leading M-by-M upper triangular part of A contains the upper triangular matrix R, and elements N-L+1 to N of the first M rows of A, with the array TAU, represent the orthogonal matrix Z as a product of M elementary reflectors. /param>
Parameters
LDA(input) INTEGER The leading dimension of the array A. LDA .GE. max(1,M). /param>
Parameters
TAU(output) DOUBLE PRECISION array, dimension (M) The scalar factors of the elementary reflectors. /param>
Parameters
WORK(workspace) DOUBLE PRECISION array, dimension (M) /param>
