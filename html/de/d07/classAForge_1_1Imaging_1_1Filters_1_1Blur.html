Detailed Description

Blur filter.

The filter performs convolution filter using the blur kernel:

1 2 3 2 1
2 4 5 4 2
3 5 6 5 3
2 4 5 4 2
1 2 3 2 1

For the list of supported pixel formats, see the documentation to Convolution filter.

<note>By default this filter sets Convolution.ProcessAlpha property to true, so the alpha channel of 32 bpp and 64 bpp images is blurred as well. </note>

Sample usage:

// create filter
Blur filter = new Blur( );
// apply the filter
filter.ApplyInPlace( image );
