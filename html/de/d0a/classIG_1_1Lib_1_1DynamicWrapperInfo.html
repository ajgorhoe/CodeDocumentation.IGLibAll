Detailed Description

Contains information about a DynamicWrapper object of specific type WrapperType .

Used as auxiliary class for manipulating objects of type WrapperType (specificallt WrapperType ), used especially by the DynamicWrapperManipulator<WrapperType>.

Inheritance from DynamicWrapper.AccessorDynamicWrapper enables access to protected and privete mebers of DynamicWrapper.

Stores additional infoo about a wrapped object or one of its nested objects in object graph, such as parent object of an object (when defined and needed in manipulation) and arbitrary dynamic info.

DynamicWrapperInfo<WrapperType>.Wrapper contains wrapped object of interest, whose information is stored. Object being wrapped enables use of field accessors, array indexers and other utilities of DynamicWrapper.

Important:

This object never contains existent wrapper objects created elsewhere. This is to prevent inadvertently wrapping a different object into existent wrapper. Containing wrappers are always created by class itself.
