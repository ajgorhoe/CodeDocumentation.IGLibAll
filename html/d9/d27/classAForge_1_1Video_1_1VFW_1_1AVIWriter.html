Detailed Description

AVI files writing using Video for Windows interface.

The class allows to write AVI files using Video for Windows API.

Sample usage:

// instantiate AVI writer, use WMV3 codec
AVIWriter writer = new AVIWriter( "wmv3" );
// create new AVI file and open it
writer.Open( "test.avi", 320, 240 );
// create frame image
Bitmap image = new Bitmap( 320, 240 );
for ( int i = 0; i < 240; i++ )
{
// update image
image.SetPixel( i, i, Color.Red );
// add the image as a new frame of video file
writer.AddFrame( image );
}
writer.Close( );

◆ Codec

Codec used for video compression.

The property sets the FOURCC code of video compression codec, which needs to be used for video encoding.

<note>The property should be set befor opening new file to take effect.</note>

Default video codec is set "DIB ", which means no compression.

◆ Quality

int AForge.Video.VFW.AVIWriter.Quality

Compression video quality.

The property sets video quality used by codec in order to balance compression rate and image quality. The quality is measured usually in the [0, 100] range.

<note>The property should be set befor opening new file to take effect.</note>

Default value is set to -1 - default compression quality of the codec.
