Detailed Description

A structure representing 3x3 matrix.

The structure incapsulates elements of a 3x3 matrix and provides some operations with it.

out float  pitch,

inline

Extract rotation angles from the rotation matrix.

Parameters
yawExtracted rotation angle around Y axis in radians.
pitchExtracted rotation angle around X axis in radians.
rollExtracted rotation angle around Z axis in radians.

<note>The routine assumes roll-pitch-yaw rotation order when extracting rotation angle. Using extracted angles with the CreateFromYawPitchRoll should provide same rotation matrix. </note>

<note>The method assumes the provided matrix represent valid rotation matrix.</note>

Sample usage:

// assume we have a rotation matrix created like this
float yaw = 10.0f / 180 * Math.PI;
float pitch = 30.0f / 180 * Math.PI;
float roll = 45.0f / 180 * Math.PI;
Matrix3x3 rotationMatrix = Matrix3x3.CreateFromYawPitchRoll( yaw, pitch, roll );
// ...
// now somewhere in the code you may want to get rotation
// angles back from a matrix assuming same rotation order
float extractedYaw;
float extractedPitch;
float extractedRoll;
rotation.ExtractYawPitchRoll( out extractedYaw, out extractedPitch, out extractedRoll );

Calculate Singular Value Decomposition (SVD) of the matrix, such as A=U*E*VT.

Parameters
uOutput parameter which gets 3x3 U matrix.
eOutput parameter which gets diagonal elements of the E matrix.
vOutput parameter which gets 3x3 V matrix.

Having components U, E and V the source matrix can be reproduced using below code:

Matrix3x3 source = u * Matrix3x3.Diagonal( e ) * v.Transpose( );
