Detailed Description

– LAPACK routine (version 3.0) – Univ. of Tennessee, Univ. of California Berkeley, NAG Ltd., Courant Institute, Argonne National Lab, and Rice University March 31, 1993

Purpose

DTRCON estimates the reciprocal of the condition number of a triangular matrix A, in either the 1-norm or the infinity-norm.

The norm of A is computed and an estimate is obtained for norm(inv(A)), then the reciprocal of the condition number is computed as RCOND = 1 / ( norm(A) * norm(inv(A)) ).

/summary>

Purpose

DTRCON estimates the reciprocal of the condition number of a triangular matrix A, in either the 1-norm or the infinity-norm.

The norm of A is computed and an estimate is obtained for norm(inv(A)), then the reciprocal of the condition number is computed as RCOND = 1 / ( norm(A) * norm(inv(A)) ).

/summary>

Parameters
NORM(input) CHARACTER*1 Specifies whether the 1-norm condition number or the infinity-norm condition number is required: = '1' or 'O': 1-norm; = 'I': Infinity-norm. /param>
Parameters
UPLO(input) CHARACTER*1 = 'U': A is upper triangular; = 'L': A is lower triangular. /param>
Parameters
DIAG(input) CHARACTER*1 = 'N': A is non-unit triangular; = 'U': A is unit triangular. /param>
Parameters
N(input) INTEGER The order of the matrix A. N .GE. 0. /param>
Parameters
A(input) DOUBLE PRECISION array, dimension (LDA,N) The triangular matrix A. If UPLO = 'U', the leading N-by-N upper triangular part of the array A contains the upper triangular matrix, and the strictly lower triangular part of A is not referenced. If UPLO = 'L', the leading N-by-N lower triangular part of the array A contains the lower triangular matrix, and the strictly upper triangular part of A is not referenced. If DIAG = 'U', the diagonal elements of A are also not referenced and are assumed to be 1. /param>
Parameters
LDA(input) INTEGER The leading dimension of the array A. LDA .GE. max(1,N). /param>
Parameters
RCOND(output) DOUBLE PRECISION The reciprocal of the condition number of the matrix A, computed as RCOND = 1/(norm(A) * norm(inv(A))). /param>
Parameters
WORK(workspace) DOUBLE PRECISION array, dimension (3*N) /param>
Parameters
IWORK(workspace) INTEGER array, dimension (N) /param>
Parameters
INFO(output) INTEGER = 0: successful exit .LT. 0: if INFO = -i, the i-th argument had an illegal value /param>
