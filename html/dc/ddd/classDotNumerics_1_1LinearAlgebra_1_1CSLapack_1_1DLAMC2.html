Purpose

DLAMC2 determines the machine parameters specified in its argument list.

/summary>

Parameters
BETA(output) INTEGER The base of the machine. /param>
Parameters
T(output) INTEGER The number of ( BETA ) digits in the mantissa. /param>
Parameters
RND(output) LOGICAL Specifies whether proper rounding ( RND = .TRUE. ) or chopping ( RND = .FALSE. ) occurs in addition. This may not be a reliable guide to the way in which the machine performs its arithmetic. /param>
Parameters
EPS(output) DOUBLE PRECISION The smallest positive number such that
fl( 1.0 - EPS ) .LT. 1.0,
where fl denotes the computed value. /param>
Parameters
EMIN(output) INTEGER The minimum exponent before (gradual) underflow occurs. /param>
Parameters
RMIN(output) DOUBLE PRECISION The smallest normalized number for the machine, given by BASE**( EMIN - 1 ), where BASE is the floating point value of BETA. /param>
Parameters
EMAX(output) INTEGER The maximum exponent before overflow occurs. /param>
Parameters
RMAX(output) DOUBLE PRECISION The largest positive number for the machine, given by BASE**EMAX * ( 1 - EPS ), where BASE is the floating point value of BETA. /param>
