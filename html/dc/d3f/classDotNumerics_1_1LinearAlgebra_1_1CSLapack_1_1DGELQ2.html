Purpose

DGELQ2 computes an LQ factorization of a real m by n matrix A: A = L * Q.

/summary>

Parameters
M(input) INTEGER The number of rows of the matrix A. M .GE. 0. /param>
Parameters
N(input) INTEGER The number of columns of the matrix A. N .GE. 0. /param>
Parameters
A(input/output) DOUBLE PRECISION array, dimension (LDA,N) On entry, the m by n matrix A. On exit, the elements on and below the diagonal of the array contain the m by min(m,n) lower trapezoidal matrix L (L is lower triangular if m .LE. n); the elements above the diagonal, with the array TAU, represent the orthogonal matrix Q as a product of elementary reflectors (see Further Details). /param>
Parameters
LDA(input) INTEGER The leading dimension of the array A. LDA .GE. max(1,M). /param>
Parameters
TAU(output) DOUBLE PRECISION array, dimension (min(M,N)) The scalar factors of the elementary reflectors (see Further Details). /param>
Parameters
WORK(workspace) DOUBLE PRECISION array, dimension (M) /param>
Parameters
INFO(output) INTEGER = 0: successful exit .LT. 0: if INFO = -i, the i-th argument had an illegal value /param>
