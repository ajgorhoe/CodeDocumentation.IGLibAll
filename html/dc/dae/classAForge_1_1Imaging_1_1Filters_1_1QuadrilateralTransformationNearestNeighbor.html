◆ AutomaticSizeCalculaton

bool AForge.Imaging.Filters.QuadrilateralTransformationNearestNeighbor.AutomaticSizeCalculaton
getset

Automatic calculation of destination image or not.

The property specifies how to calculate size of destination (transformed) image. If the property is set to false, then NewWidth and NewHeight properties have effect and destination image's size is specified by user. If the property is set to true, then setting the above mentioned properties does not have any effect, but destionation image's size is automatically calculated from SourceCorners property - width and height come from length of longest edges.
