Detailed Description

– LAPACK routine (version 3.1) – Univ. of Tennessee, Univ. of California Berkeley and NAG Ltd.. November 2006

Purpose

DTRTRS solves a triangular system of the form

A * X = B or A**T * X = B,

where A is a triangular matrix of order N, and B is an N-by-NRHS matrix. A check is made to verify that A is nonsingular.

/summary>

Purpose

DTRTRS solves a triangular system of the form

A * X = B or A**T * X = B,

where A is a triangular matrix of order N, and B is an N-by-NRHS matrix. A check is made to verify that A is nonsingular.

/summary>

Parameters
UPLO(input) CHARACTER*1 = 'U': A is upper triangular; = 'L': A is lower triangular. /param>
Parameters
TRANS(input) CHARACTER*1 Specifies the form of the system of equations: = 'N': A * X = B (No transpose) = 'T': A**T * X = B (Transpose) = 'C': A**H * X = B (Conjugate transpose = Transpose) /param>
Parameters
DIAG(input) CHARACTER*1 = 'N': A is non-unit triangular; = 'U': A is unit triangular. /param>
Parameters
N(input) INTEGER The order of the matrix A. N .GE. 0. /param>
Parameters
NRHS(input) INTEGER The number of right hand sides, i.e., the number of columns of the matrix B. NRHS .GE. 0. /param>
Parameters
A
  • X = B or A**T * X = B, /param>
    Parameters
    LDA(input) INTEGER The leading dimension of the array A. LDA .GE. max(1,N). /param>
    Parameters
    B(input/output) DOUBLE PRECISION array, dimension (LDB,NRHS) On entry, the right hand side matrix B. On exit, if INFO = 0, the solution matrix X. /param>
    Parameters
    LDB(input) INTEGER The leading dimension of the array B. LDB .GE. max(1,N). /param>
    Parameters
    INFO(output) INTEGER = 0: successful exit .LT. 0: if INFO = -i, the i-th argument had an illegal value .GT. 0: if INFO = i, the i-th diagonal element of A is zero, indicating that the matrix is singular and the solutions X have not been computed. /param>
