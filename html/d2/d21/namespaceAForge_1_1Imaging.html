enum AForge.Imaging.ObjectsOrder
strong

Possible object orders.

The enumeration defines possible sorting orders of objects, found by blob counting classes.

Enumerator
None 

Unsorted order (as it is collected by algorithm).

Size 

Objects are sorted by size in descending order (bigger objects go first). Size is calculated as Width * Height.

Area 

Objects are sorted by area in descending order (bigger objects go first).

YX 

Objects are sorted by Y coordinate, then by X coordinate in ascending order (smaller coordinates go first).

XY 

Objects are sorted by X coordinate, then by Y coordinate in ascending order (smaller coordinates go first).
