◆ OriginalBitsPerPixl

int AForge.Imaging.Formats.FITSImageInfo.OriginalBitsPerPixl
getset

Original bits per pixel.

The property specifies original number of bits per image's pixel. For FITS images the value may be equal to 8, 16, 32, -32 (32 bit image with float data type for pixel encoding), -64 (64 bit image with double data type for pixel encoding).

Referenced by AForge.Imaging.Formats.FITSCodec.ReadHeader(), and AForge.Imaging.Formats.FITSCodec.ReadImageFrame().

double AForge.Imaging.Formats.FITSImageInfo.MinDataValue

Minimum data value found during parsing FITS image.

Minimum and maximum data values are used to scale image's data converting them from original bits per pixel format to supported bits per pixel format.
