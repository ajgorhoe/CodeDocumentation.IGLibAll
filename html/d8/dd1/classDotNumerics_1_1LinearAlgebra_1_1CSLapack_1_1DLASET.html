Purpose

DLASET initializes an m-by-n matrix A to BETA on the diagonal and ALPHA on the offdiagonals.

/summary>

Parameters
UPLO(input) CHARACTER*1 Specifies the part of the matrix A to be set. = 'U': Upper triangular part is set; the strictly lower triangular part of A is not changed. = 'L': Lower triangular part is set; the strictly upper triangular part of A is not changed. Otherwise: All of the matrix A is set. /param>
Parameters
M(input) INTEGER The number of rows of the matrix A. M .GE. 0. /param>
Parameters
N(input) INTEGER The number of columns of the matrix A. N .GE. 0. /param>
Parameters
ALPHA(input) DOUBLE PRECISION The constant to which the offdiagonal elements are to be set. /param>
Parameters
BETA(input) DOUBLE PRECISION The constant to which the diagonal elements are to be set. /param>
Parameters
A(input/output) DOUBLE PRECISION array, dimension (LDA,N) On exit, the leading m-by-n submatrix of A is set as follows:
if UPLO = 'U', A(i,j) = ALPHA, 1.LE.i.LE.j-1, 1.LE.j.LE.n, if UPLO = 'L', A(i,j) = ALPHA, j+1.LE.i.LE.m, 1.LE.j.LE.n, otherwise, A(i,j) = ALPHA, 1.LE.i.LE.m, 1.LE.j.LE.n, i.ne.j,
and, for all UPLO, A(i,i) = BETA, 1.LE.i.LE.min(m,n). /param>
Parameters
LDA(input) INTEGER The leading dimension of the array A. LDA .GE. max(1,M). /param>
