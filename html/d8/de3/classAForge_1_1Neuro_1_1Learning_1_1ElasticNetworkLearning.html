◆ Run() [3/3]

double AForge.Neuro.Learning.ElasticNetworkLearning.Run ( double[]  input)
inline

Runs learning iteration.

Parameters
inputInput vector.
Returns
Returns learning error - summary absolute difference between neurons' weights and appropriate inputs. The difference is measured according to the neurons distance to the winner neuron.

The method runs one learning iterations - finds winner neuron (the neuron which has weights with values closest to the specified input vector) and updates its weight (as well as weights of neighbor neurons) in the way to decrease difference with the specified input vector.

Implements AForge.Neuro.Learning.IUnsupervisedLearning.

Learning radius

Learning radius, [0, 1].

Determines the amount of neurons to be updated around winner neuron. Neurons, which are in the circle of specified radius, are updated during the learning procedure. Neurons, which are closer to the winner neuron, get more update.

Default value equals to 0.5.
