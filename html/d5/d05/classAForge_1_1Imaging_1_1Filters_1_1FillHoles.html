Detailed Description

Fill holes in objects in binary image.

The filter allows to fill black holes in white object in a binary image. It is possible to specify maximum holes' size to fill using MaxHoleWidth and MaxHoleHeight properties.

The filter accepts binary image only, which are represented as 8 bpp images.

Sample usage:

// create and configure the filter
FillHoles filter = new FillHoles( );
filter.MaxHoleHeight = 20;
filter.MaxHoleWidth = 20;
filter.CoupledSizeFiltering = false;
// apply the filter
Bitmap result = filter.Apply( image );

◆ CoupledSizeFiltering

bool AForge.Imaging.Filters.FillHoles.CoupledSizeFiltering
getset

Specifies if size filetering should be coupled or not.

In uncoupled filtering mode, holes are filled in the case if their width is smaller than or equal to MaxHoleWidth or height is smaller than or equal to MaxHoleHeight. But in coupled filtering mode, holes are filled only in the case if both width and height are smaller or equal to the corresponding value.

Default value is set to true, what means coupled filtering by size.

Maximum width of a hole to fill.

All holes, which have width greater than this value, are kept unfilled. See CoupledSizeFiltering for additional information.
