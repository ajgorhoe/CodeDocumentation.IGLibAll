Detailed Description

– LAPACK driver routine (version 3.1) – Univ. of Tennessee, Univ. of California Berkeley and NAG Ltd.. November 2006

Purpose

DLANV2 computes the Schur factorization of a real 2-by-2 nonsymmetric matrix in standard form:

[ A B ] = [ CS -SN ] [ AA BB ] [ CS SN ] [ C D ] [ SN CS ] [ CC DD ] [-SN CS ]

where either 1) CC = 0 so that AA and DD are real eigenvalues of the matrix, or 2) AA = DD and BB*CC .LT. 0, so that AA + or - sqrt(BB*CC) are complex conjugate eigenvalues.

/summary>
