Purpose

DLAMC1 determines the machine parameters given by BETA, T, RND, and IEEE1.

/summary>

Parameters
BETA(output) INTEGER The base of the machine. /param>
Parameters
T(output) INTEGER The number of ( BETA ) digits in the mantissa. /param>
Parameters
RND(output) LOGICAL Specifies whether proper rounding ( RND = .TRUE. ) or chopping ( RND = .FALSE. ) occurs in addition. This may not be a reliable guide to the way in which the machine performs its arithmetic. /param>
Parameters
IEEE1(output) LOGICAL Specifies whether rounding appears to be done in the IEEE 'round to nearest' style. /param>
