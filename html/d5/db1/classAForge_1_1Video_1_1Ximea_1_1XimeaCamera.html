Detailed Description

The class provides access to XIMEA cameras.

The class allows to perform image acquisition from XIMEA cameras. It wraps XIMEA'a xiAPI, which means that users of this class will also require m3api.dll and a correct TM file for the camera model connected to the system (both are provided with XIMEA API software package).

Sample usage:

XimeaCamera camera = new XimeaCamera( );
// open camera and start data acquisition
camera.Open( 0 );
camera.StartAcquisition( );
// set exposure time to 10 milliseconds
camera.SetParam( CameraParameter.Exposure, 10 * 1000 );
// get image from the camera
Bitmap bitmap = camera.GetImage( );
// process the image
// ...
// dispose the image when it is no longer needed
bitmap.Dispose( );
// stop data acquisition and close the camera
camera.StopAcquisition( );
camera.Close( );

Bitmap AForge.Video.Ximea.XimeaCamera.GetImage ( int  timeout,
bool  makeCopy 
)
inline

Get image from the opened XIMEA camera.

Parameters
timeoutMaximum time to wait in milliseconds till image becomes available.
makeCopyMake a copy of the camera's image or not.
Returns
Returns image retrieved from the camera.

If the makeCopy is set to true, then the method creates a managed copy of the camera's image, so the managed image stays valid even when the camera is closed. However, setting this parameter to false creates a managed image which is just a wrapper around camera's unmanaged image. So if camera is closed and its resources are freed, the managed image becomes no longer valid and accessing it will generate an exception.

Exceptions
VideoExceptionAn error occurred while communicating with a camera. See error message for additional information.
NotConnectedExceptionNo camera was opened, so can not access its methods.
TimeoutExceptionTime out value reached - no image is available within specified time value.
