Detailed Description

Invert image.

The filter inverts colored and grayscale images.

The filter accepts 8, 16 bpp grayscale and 24, 48 bpp color images for processing.

Sample usage:

// create filter
Invert filter = new Invert( );
// apply the filter
filter.ApplyInPlace( image );
