Detailed Description

A window form to calculate various checksums of selected files or inserted text.

This is only a window container for HashForm.

See:

http://www2.arnes.si/~ljc3m2/igor/software/IGLibShellApp/HashForm.html#hashform

$A Igor Aug 08;
