Detailed Description

Base class for storing file- or directory- related data.

Main purpose is to serve as base class for FileDataBaseObsolete and DirectoryDataBase

This class provides definitions that are common for file and directory data classes.

For this and related classes, basic path information consists of file or directory name, and either parent directory absolute path or base path plus parent directory relative path. In this way, path is provided in dcomposed way.

Derived path information is always calculated on demand and is cached

◆ Exists

bool IG.Lib.FileOrDirectoryDataBase.Exists
getset

Whether the file/directory exists. Dafault is false.

This property is nullable, null meaning that there is no information.

File or directory data object can be manipulated if the corresponding file or directory does not existt (i.e., to store informaton about previous state of apart of file system). This flag is used to specify what is actually the case (whether file or directory actually exists) but may not be used in some scenarios.
