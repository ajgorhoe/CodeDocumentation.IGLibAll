Detailed Description

– LAPACK routine (version 3.1) – Univ. of Tennessee, Univ. of California Berkeley and NAG Ltd.. November 2006

Purpose

DGETRI computes the inverse of a matrix using the LU factorization computed by DGETRF.

This method inverts U and then computes inv(A) by solving the system inv(A)*L = inv(U) for inv(A).

/summary>

Purpose

DGETRI computes the inverse of a matrix using the LU factorization computed by DGETRF.

This method inverts U and then computes inv(A) by solving the system inv(A)*L = inv(U) for inv(A).

/summary>

Parameters
N(input) INTEGER The order of the matrix A. N .GE. 0. /param>
Parameters
A(input/output) DOUBLE PRECISION array, dimension (LDA,N) On entry, the factors L and U from the factorization A = P*L*U as computed by DGETRF. On exit, if INFO = 0, the inverse of the original matrix A. /param>
Parameters
LDA(input) INTEGER The leading dimension of the array A. LDA .GE. max(1,N). /param>
Parameters
IPIV(input) INTEGER array, dimension (N) The pivot indices from DGETRF; for 1.LE.i.LE.N, row i of the matrix was interchanged with row IPIV(i). /param>
Parameters
WORK(workspace/output) DOUBLE PRECISION array, dimension (MAX(1,LWORK)) On exit, if INFO=0, then WORK(1) returns the optimal LWORK. /param>
Parameters
LWORK(input) INTEGER The dimension of the array WORK. LWORK .GE. max(1,N). For optimal performance LWORK .GE. N*NB, where NB is the optimal blocksize returned by ILAENV.
If LWORK = -1, then a workspace query is assumed; the routine only calculates the optimal size of the WORK array, returns this value as the first entry of the WORK array, and no error message related to LWORK is issued by XERBLA. /param>
Parameters
INFO(output) INTEGER = 0: successful exit .LT. 0: if INFO = -i, the i-th argument had an illegal value .GT. 0: if INFO = i, U(i,i) is exactly zero; the matrix is singular and its inverse could not be computed. /param>
