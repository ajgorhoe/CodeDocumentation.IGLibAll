Purpose

DLAQP2 computes a QR factorization with column pivoting of the block A(OFFSET+1:M,1:N). The block A(1:OFFSET,1:N) is accordingly pivoted, but not factorized.

/summary>

Parameters
M(input) INTEGER The number of rows of the matrix A. M .GE. 0. /param>
Parameters
N(input) INTEGER The number of columns of the matrix A. N .GE. 0. /param>
Parameters
OFFSET(input) INTEGER The number of rows of the matrix A that must be pivoted but no factorized. OFFSET .GE. 0. /param>
Parameters
A(input/output) DOUBLE PRECISION array, dimension (LDA,N) On entry, the M-by-N matrix A. On exit, the upper triangle of block A(OFFSET+1:M,1:N) is the triangular factor obtained; the elements in block A(OFFSET+1:M,1:N) below the diagonal, together with the array TAU, represent the orthogonal matrix Q as a product of elementary reflectors. Block A(1:OFFSET,1:N) has been accordingly pivoted, but no factorized. /param>
Parameters
LDA(input) INTEGER The leading dimension of the array A. LDA .GE. max(1,M). /param>
Parameters
JPVT(input/output) INTEGER array, dimension (N) On entry, if JPVT(i) .ne. 0, the i-th column of A is permuted to the front of A*P (a leading column); if JPVT(i) = 0, the i-th column of A is a free column. On exit, if JPVT(i) = k, then the i-th column of A*P was the k-th column of A. /param>
Parameters
TAU(output) DOUBLE PRECISION array, dimension (min(M,N)) The scalar factors of the elementary reflectors. /param>
Parameters
VN1(input/output) DOUBLE PRECISION array, dimension (N) The vector with the partial column norms. /param>
Parameters
VN2(input/output) DOUBLE PRECISION array, dimension (N) The vector with the exact column norms. /param>
Parameters
WORK(workspace) DOUBLE PRECISION array, dimension (N) /param>
