Detailed Description

– LAPACK auxiliary routine (version 3.1) – Univ. of Tennessee, Univ. of California Berkeley and NAG Ltd.. November 2006

Purpose

DLAEXC swaps adjacent diagonal blocks T11 and T22 of order 1 or 2 in an upper quasi-triangular matrix T by an orthogonal similarity transformation.

T must be in Schur canonical form, that is, block upper triangular with 1-by-1 and 2-by-2 diagonal blocks; each 2-by-2 diagonal block has its diagonal elemnts equal and its off-diagonal elements of opposite sign.

/summary>

Purpose

DLAEXC swaps adjacent diagonal blocks T11 and T22 of order 1 or 2 in an upper quasi-triangular matrix T by an orthogonal similarity transformation.

T must be in Schur canonical form, that is, block upper triangular with 1-by-1 and 2-by-2 diagonal blocks; each 2-by-2 diagonal block has its diagonal elemnts equal and its off-diagonal elements of opposite sign.

/summary>

Parameters
WANTQ(input) LOGICAL = .TRUE. : accumulate the transformation in the matrix Q; = .FALSE.: do not accumulate the transformation. /param>
Parameters
N(input) INTEGER The order of the matrix T. N .GE. 0. /param>
Parameters
Tmust be in Schur canonical form, that is, block upper triangular /param>
Parameters
LDT(input) INTEGER The leading dimension of the array T. LDT .GE. max(1,N). /param>
Parameters
Q(input/output) DOUBLE PRECISION array, dimension (LDQ,N) On entry, if WANTQ is .TRUE., the orthogonal matrix Q. On exit, if WANTQ is .TRUE., the updated matrix Q. If WANTQ is .FALSE., Q is not referenced. /param>
Parameters
LDQ(input) INTEGER The leading dimension of the array Q. LDQ .GE. 1; and if WANTQ is .TRUE., LDQ .GE. N. /param>
Parameters
J1(input) INTEGER The index of the first row of the first block T11. /param>
Parameters
N1(input) INTEGER The order of the first block T11. N1 = 0, 1 or 2. /param>
Parameters
N2(input) INTEGER The order of the second block T22. N2 = 0, 1 or 2. /param>
Parameters
WORK(workspace) DOUBLE PRECISION array, dimension (N) /param>
Parameters
INFO(output) INTEGER = 0: successful exit = 1: the transformed matrix T would be too far from Schur form; the blocks are not swapped and T and Q are unchanged. /param>
