Detailed Description

Drawing primitives.

The class allows to do drawing of some primitives directly on locked image data or unmanaged image.

<note>All methods of this class support drawing only on color 24/32 bpp images and on grayscale 8 bpp indexed images.</note>

<note>When it comes to alpha blending for 24/32 bpp images, all calculations are done as described on Wikipeadia (see "over" operator).</note>
