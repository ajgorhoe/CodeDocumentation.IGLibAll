Detailed Description

– LAPACK routine (version 3.1) – Univ. of Tennessee, Univ. of California Berkeley and NAG Ltd.. November 2006

Purpose

DORG2R generates an m by n real matrix Q with orthonormal columns, which is defined as the first n columns of a product of k elementary reflectors of order m

Q = H(1) H(2) . . . H(k)

as returned by DGEQRF.

/summary>

Purpose

DORG2R generates an m by n real matrix Q with orthonormal columns, which is defined as the first n columns of a product of k elementary reflectors of order m

Q = H(1) H(2) . . . H(k)

as returned by DGEQRF.

/summary>

Parameters
M(input) INTEGER The number of rows of the matrix Q. M .GE. 0. /param>
Parameters
N(input) INTEGER The number of columns of the matrix Q. M .GE. N .GE. 0. /param>
Parameters
K(input) INTEGER The number of elementary reflectors whose product defines the matrix Q. N .GE. K .GE. 0. /param>
Parameters
A(input/output) DOUBLE PRECISION array, dimension (LDA,N) On entry, the i-th column must contain the vector which defines the elementary reflector H(i), for i = 1,2,...,k, as returned by DGEQRF in the first k columns of its array argument A. On exit, the m-by-n matrix Q. /param>
Parameters
LDA(input) INTEGER The first dimension of the array A. LDA .GE. max(1,M). /param>
Parameters
TAU(input) DOUBLE PRECISION array, dimension (K) TAU(i) must contain the scalar factor of the elementary reflector H(i), as returned by DGEQRF. /param>
Parameters
WORK(workspace) DOUBLE PRECISION array, dimension (N) /param>
Parameters
INFO(output) INTEGER = 0: successful exit .LT. 0: if INFO = -i, the i-th argument has an illegal value /param>
