Detailed Description

Gaussian function.

The class is used to calculate 1D and 2D Gaussian functions for specified Sigma (s) value:

1-D: f(x) = exp( x * x / ( -2 * s * s ) ) / ( s * sqrt( 2 * PI ) )
2-D: f(x, y) = exp( x * x + y * y / ( -2 * s * s ) ) / ( s * s * 2 * PI )
