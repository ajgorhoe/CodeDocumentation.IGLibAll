Detailed Description

Histogram control.

The control displays histograms represented with integer arrays, where each array's element keeps occurrence number of the corresponding element.

Sample usage:

// create array with histogram values
int[] histogramValues = new int[] { 3, 8, 53, 57, 79, 69, ... };
// set values to histogram control
histogram.Values = histogramValues;

◆ IsLogarithmicView

bool AForge.Controls.Histogram.IsLogarithmicView
getset

Logarithmic view or not.

In the case if logarihmic view is selected, then the control will display base 10 logarithm of values.

By default the property is set to false - none logarithmic view.

◆ IsVertical

Vertical view or not.

Event Documentation

◆ PositionChanged

HistogramEventHandler AForge.Controls.Histogram.PositionChanged

Mouse position changed event.

The event is fired only if the AllowSelection property is set to true. The passed to event handler HistogramEventArgs class is initialized with HistogramEventArgs.Position property only, which is histogram value's index pointed by mouse.

◆ SelectionChanged

HistogramEventHandler AForge.Controls.Histogram.SelectionChanged

Mouse selection changed event.

The event is fired only if the AllowSelection property is set to true. The passed to event handler HistogramEventArgs class is initialized with HistogramEventArgs.Min and HistogramEventArgs.Max properties only, which represent selection range - min and max indexes.
