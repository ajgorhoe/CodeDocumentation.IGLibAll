Detailed Description

Hue picker control.

The control allows selecting hue value (or range) from HSL color space. Hue values are integer values in the [0, 359] range.

Member Enumeration Documentation

◆ HuePickerType

enum AForge.Controls.HuePicker.HuePickerType
strong

Enumeration of hue picker types.

The HuePickerType.Value type provides single bullet to drag, which allows selecting single hue value. The value is accessible through Value property.

The HuePickerType.Range type provides two bullets to drag, which correspond to minimum and maximum values of the hue range. These values are accessible through Min and Max properties.

Enumerator
Value 

Selecting single hue value.

Range
