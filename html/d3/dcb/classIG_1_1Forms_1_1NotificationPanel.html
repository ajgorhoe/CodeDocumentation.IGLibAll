Sets the parameters of the current control according to the specified NotificationParameters object plus according to explicitly set parameters.

First, parameters are set according to contents of another object of type NotificationParameters, and then parameters that are explicitly specified are set to specified values.

This makes possible to ues one tamplate object whith which parameeterrs can be re-set to some reference state, while explicitly specifying those parameters that should be set differently from this.

Parameters
parParameters object from which all parameters are first copied (and eventually overwritten later according to explicitly set parameters).

If null then default parameter object DefaultNotificationParameters is taken, i.e., parameters are set to default values.

Parameters
titleTitle of notification.

null means that title is not set here (use empty string "" for no title). In this case, title is still set if type is defined.

Title can not be omitted, the reason is that this function with only par specified must be distinguishable from SetParameters(NotificationParameters).

Parameters
notificationTypeType of notification (e.g. error, warning, info...), see ReportType.
showTimeTime span during which notification is visibly displayed (it disappears from display after this time).
fadingPortionPortion of time at the end of the display time (showTime) during which notification fades out.
timerIntervalInterval between timer events being fired. This defines e.g. how frequently background color is changed during fadeout.
backgroundColorBackground color. If null or not specified, then it is not modified, unless the
fadeColorBackground color after notification fades out (during fadeout, color changes linearly from initial background to this color).
titleColorTitle color.
messageColorMassage color.

NotificationParameters IG.Forms.NotificationPanel.Parameters
getprotected

Internal parameters of the current NotificationPanel.

This object leverages functionality of NotificationParameters class in setting internal parameters.

Its parameters are initially set to default values via template NotificationParameters.DefaultNotificationParameters.

This object is used as primary storage for parameters. Individual properties reference properties of this object and are used in order to set parameters in GUI builders.

◆ Title

string IG.Forms.NotificationPanel.Title
getset

Message title.

Title is included in parameters because it can be set automatically when not specified.

If null title is provided in constructor or SetParameters(NotificationParameters) then title is set automatically.

If empty string is provided then it is considered that there is no title.
