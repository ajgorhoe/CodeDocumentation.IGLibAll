Detailed Description

The class provides support for parallel computations, paralleling loop's iterations.

The class allows to parallel loop's iteration computing them in separate threads, what allows their simultaneous execution on multiple CPUs/cores.

Executes a for-loop in which iterations may run in parallel.

Parameters
startLoop's start index.
stopLoop's stop index.
loopBodyLoop's body.

The method is used to parallel for-loop running its iterations in different threads. The start and stop parameters define loop's starting and ending loop's indexes. The number of iterations is equal to stop - start.

Sample usage:

Parallel.For( 0, 20, delegate( int i )
// which is equivalent to
// for ( int i = 0; i < 20; i++ )
{
System.Diagnostics.Debug.WriteLine( "Iteration: " + i );
// ...
} );

int AForge.Parallel.ThreadsCount
staticgetset

Number of threads used for parallel computations.

The property sets how many worker threads are created for paralleling loops' computations.

By default the property is set to number of CPU's in the system (see System.Environment.ProcessorCount).
