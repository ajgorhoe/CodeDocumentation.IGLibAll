Detailed Description

– LAPACK auxiliary routine (version 3.1) – Univ. of Tennessee, Univ. of California Berkeley and NAG Ltd.. November 2006

Purpose

DLASY2 solves for the N1 by N2 matrix X, 1 .LE. N1,N2 .LE. 2, in

op(TL)*X + ISGN*X*op(TR) = SCALE*B,

where TL is N1 by N1, TR is N2 by N2, B is N1 by N2, and ISGN = 1 or -1. op(T) = T or T', where T' denotes the transpose of T.

Purpose

DLASY2 solves for the N1 by N2 matrix X, 1 .LE. N1,N2 .LE. 2, in

op(TL)*X + ISGN*X*op(TR) = SCALE*B,

where TL is N1 by N1, TR is N2 by N2, B is N1 by N2, and ISGN = 1 or -1. op(T) = T or T', where T' denotes the transpose of T.

/summary>

Parameters
LTRANL(input) LOGICAL On entry, LTRANL specifies the op(TL): = .FALSE., op(TL) = TL, = .TRUE., op(TL) = TL'. /param>
Parameters
LTRANR(input) LOGICAL On entry, LTRANR specifies the op(TR): = .FALSE., op(TR) = TR, = .TRUE., op(TR) = TR'. /param>
Parameters
ISGN(input) INTEGER On entry, ISGN specifies the sign of the equation as described before. ISGN may only be 1 or -1. /param>
Parameters
N1(input) INTEGER On entry, N1 specifies the order of matrix TL. N1 may only be 0, 1 or 2. /param>
Parameters
N2(input) INTEGER On entry, N2 specifies the order of matrix TR. N2 may only be 0, 1 or 2. /param>
Parameters
TL(input) DOUBLE PRECISION array, dimension (LDTL,2) On entry, TL contains an N1 by N1 matrix. /param>
Parameters
LDTL(input) INTEGER The leading dimension of the matrix TL. LDTL .GE. max(1,N1). /param>
Parameters
TR(input) DOUBLE PRECISION array, dimension (LDTR,2) On entry, TR contains an N2 by N2 matrix. /param>
Parameters
LDTR(input) INTEGER The leading dimension of the matrix TR. LDTR .GE. max(1,N2). /param>
Parameters
B(input) DOUBLE PRECISION array, dimension (LDB,2) On entry, the N1 by N2 matrix B contains the right-hand side of the equation. /param>
Parameters
LDB(input) INTEGER The leading dimension of the matrix B. LDB .GE. max(1,N1). /param>
Parameters
SCALE(output) DOUBLE PRECISION On exit, SCALE contains the scale factor. SCALE is chosen less than or equal to 1 to prevent the solution overflowing. /param>
Parameters
X(output) DOUBLE PRECISION array, dimension (LDX,2) On exit, X contains the N1 by N2 solution. /param>
Parameters
LDX(input) INTEGER The leading dimension of the matrix X. LDX .GE. max(1,N1). /param>
Parameters
XNORM(output) DOUBLE PRECISION On exit, XNORM is the infinity-norm of the solution. /param>
Parameters
INFO(output) INTEGER On exit, INFO is set to 0: successful exit. 1: TL and TR have too close eigenvalues, so TL or TR is perturbed to get a nonsingular equation. NOTE: In the interests of speed, this routine does not check the inputs for errors. /param>
