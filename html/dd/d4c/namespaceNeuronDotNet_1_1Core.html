enum NeuronDotNet.Core.ConnectionMode
strong

Mode of connection between layers.

Enumerator
Complete 

A connection mode where all neurons of source layer are connected to all neurons of target layer

OneOne 

A connection mode where each neuron in source layer is connected to a single distinct neuron in the target layer. The source and target layers should have same number of neurons.
