Purpose

DGBTRS solves a system of linear equations A * X = B or A' * X = B with a general band matrix A using the LU factorization computed by DGBTRF.

/summary>

Parameters
TRANS(input) CHARACTER*1 Specifies the form of the system of equations. = 'N': A * X = B (No transpose) = 'T': A'* X = B (Transpose) = 'C': A'* X = B (Conjugate transpose = Transpose) /param>
Parameters
N(input) INTEGER The order of the matrix A. N .GE. 0. /param>
Parameters
KL(input) INTEGER The number of subdiagonals within the band of A. KL .GE. 0. /param>
Parameters
KU(input) INTEGER The number of superdiagonals within the band of A. KU .GE. 0. /param>
Parameters
NRHS(input) INTEGER The number of right hand sides, i.e., the number of columns of the matrix B. NRHS .GE. 0. /param>
Parameters
AB(input) DOUBLE PRECISION array, dimension (LDAB,N) Details of the LU factorization of the band matrix A, as computed by DGBTRF. U is stored as an upper triangular band matrix with KL+KU superdiagonals in rows 1 to KL+KU+1, and the multipliers used during the factorization are stored in rows KL+KU+2 to 2*KL+KU+1. /param>
Parameters
LDAB(input) INTEGER The leading dimension of the array AB. LDAB .GE. 2*KL+KU+1. /param>
Parameters
IPIV(input) INTEGER array, dimension (N) The pivot indices; for 1 .LE. i .LE. N, row i of the matrix was interchanged with row IPIV(i). /param>
Parameters
B(input/output) DOUBLE PRECISION array, dimension (LDB,NRHS) On entry, the right hand side matrix B. On exit, the solution matrix X. /param>
Parameters
LDB(input) INTEGER The leading dimension of the array B. LDB .GE. max(1,N). /param>
Parameters
INFO(output) INTEGER = 0: successful exit .LT. 0: if INFO = -i, the i-th argument had an illegal value /param>
