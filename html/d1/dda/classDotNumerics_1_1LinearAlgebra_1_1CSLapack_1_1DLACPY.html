Purpose

DLACPY copies all or part of a two-dimensional matrix A to another matrix B.

/summary>

Parameters
UPLO(input) CHARACTER*1 Specifies the part of the matrix A to be copied to B. = 'U': Upper triangular part = 'L': Lower triangular part Otherwise: All of the matrix A /param>
Parameters
M(input) INTEGER The number of rows of the matrix A. M .GE. 0. /param>
Parameters
N(input) INTEGER The number of columns of the matrix A. N .GE. 0. /param>
Parameters
A(input) DOUBLE PRECISION array, dimension (LDA,N) The m by n matrix A. If UPLO = 'U', only the upper triangle or trapezoid is accessed; if UPLO = 'L', only the lower triangle or trapezoid is accessed. /param>
Parameters
LDA(input) INTEGER The leading dimension of the array A. LDA .GE. max(1,M). /param>
Parameters
B(output) DOUBLE PRECISION array, dimension (LDB,N) On exit, B = A in the locations specified by UPLO. /param>
Parameters
LDB(input) INTEGER The leading dimension of the array B. LDB .GE. max(1,M). /param>
