Detailed Description

Local video device selection form.

The form provides a standard way of selecting local video device (USB web camera, capture board, etc. - anything supporting DirectShow interface), which can be reused across applications. It allows selecting video device, video size and snapshots size (if device supports snapshots and user needs them).

◆ ConfigureSnapshots

bool AForge.Video.DirectShow.VideoCaptureDeviceForm.ConfigureSnapshots
getset

Specifies if snapshot configuration should be done or not.

The property specifies if the dialog form should allow configuration of snapshot sizes (if selected video source supports snapshots). If the property is set to true, then the form will provide additional combo box enumerating supported snapshot sizes. Otherwise the combo boxes will be hidden.

If the property is set to true and selected device supports snapshots, then VideoCaptureDevice.ProvideSnapshots property of the configured device is set to true.

Default value of the property is set to false.
