Detailed Description

Dynamic object wrapper class for handling JSON serialization and deserialization of objects, addressing of object via field access and indexing notations, transversal of objects, performing operations on objects and their parts, and as base class to derive object that represent arbitrary tree structures such as DOMs or file structures.

This class is especially suitable for representation of objects where direct mapping between JSON and object graphs exist.

A number of standard operations are implemented such as indexing of table elements and setting of object fields vie member operator (.), JSON serialization / deserialization, wrapping and unwrapping, transversal, etc. Functionality can be extended in subclasses.
