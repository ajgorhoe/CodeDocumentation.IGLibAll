Detailed Description

Interface for custom blobs' filters used for filtering blobs after blob counting.

The interface should be implemented by classes, which perform custom blobs' filtering different from default filtering implemented in BlobCounterBase. See BlobCounterBase.BlobsFilter for additional information.
