Detailed Description

Base class for one dimensional function optimizations.

The class is aimed to be used for one dimensional function optimization problems. It implements all methods of IFitnessFunction interface and requires overriding only one method - OptimizationFunction, which represents the function to optimize.

<note>The optimization function should be greater than 0 on the specified optimization range.</note>

The class works only with binary chromosomes (BinaryChromosome).

Sample usage:

// define optimization function
public class UserFunction : OptimizationFunction1D
{
public UserFunction( ) :
base( new Range( 0, 255 ) ) { }
public override double OptimizationFunction( double x )
{
return Math.Cos( x / 23 ) * Math.Sin( x / 50 ) + 2;
}
}
...
// create genetic population
Population population = new Population( 40,
new BinaryChromosome( 32 ),
new UserFunction( ),
new EliteSelection( ) );
while ( true )
{
// run one epoch of the population
population.RunEpoch( );
// ...
}
