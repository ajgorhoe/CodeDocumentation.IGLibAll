Detailed Description

Slider control.

The control represents a slider, which can be dragged in the [-1, 1] range. Default position of the slider is set 0, which corresponds to center of the control.

◆ ResetPositionOnMouseRelease

bool AForge.Controls.SliderControl.ResetPositionOnMouseRelease
getset

Determines behaviour of manipulator, when mouse button is released.

The property controls behaviour of manipulator on releasing mouse button. If the property is set to true, then position of manipulator is reset to 0, when mouse button is released. Otherwise manipulator stays on the place, where it was left.

Default value is set to true.
