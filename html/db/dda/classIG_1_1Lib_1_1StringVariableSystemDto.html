Detailed Description

Generic class for various DTOs (Data Transfer Objects) for systems of string-valued variables. Used to store a state of a variable system or to store a set of variables to be imported into a system.

DTO (data transfer object) for String variable systems.

Template Parameters
VariableSystemTypeType parameter specifying the specific variable system type for which concrete DTO is designed.
