Detailed Description

– LAPACK auxiliary routine (version 3.0) – Univ. of Tennessee, Univ. of California Berkeley, NAG Ltd., Courant Institute, Argonne National Lab, and Rice University February 29, 1992

Purpose

DLACON estimates the 1-norm of a square, real matrix A. Reverse communication is used for evaluating matrix-vector products.

Purpose

DLACON estimates the 1-norm of a square, real matrix A. Reverse communication is used for evaluating matrix-vector products.

/summary>

Parameters
N(input) INTEGER The order of the matrix. N .GE. 1. /param>
Parameters
V(workspace) DOUBLE PRECISION array, dimension (N) On the final return, V = A*W, where EST = norm(V)/norm(W) (W is not returned). /param>
Parameters
X(input/output) DOUBLE PRECISION array, dimension (N) On an intermediate return, X should be overwritten by A * X, if KASE=1, A' * X, if KASE=2, and DLACON must be re-called with all the other parameters unchanged. /param>
Parameters
ISGN(workspace) INTEGER array, dimension (N) /param>
Parameters
EST(output) DOUBLE PRECISION An estimate (a lower bound) for norm(A). /param>
Parameters
KASE(input/output) INTEGER On the initial call to DLACON, KASE should be 0. On an intermediate return, KASE will be 1 or 2, indicating whether X should be overwritten by A * X or A' * X. On the final return from DLACON, KASE will again be 0. /param>
